# 🎯 Jogo de Adivinhação em Java

Projeto simples desenvolvido em **Java** como atividade do curso de **Desenvolvimento Back-End com Java** da Alura.

## 📌 Descrição

O programa simula um **jogo de adivinhação**, onde o sistema gera um número aleatório entre **0 e 50**.
O usuário tem **5 tentativas** para acertar o número secreto.

A cada tentativa, o programa informa se o número digitado é **maior ou menor** que o número gerado.

## 🛠 Tecnologias utilizadas

* Java
* Biblioteca `Random`
* Biblioteca `Scanner`

## ▶️ Como executar o projeto

1. Clone o repositório:

```
git clone https://github.com/devdosAnjos7/Jogo_Adivinha.git
```

2. Abra o projeto em uma IDE Java (IntelliJ, Eclipse ou VS Code).

3. Execute o arquivo:

```
Main.java
```

4. Digite os números no terminal para tentar adivinhar o número secreto.

## 📚 Objetivo do projeto

Praticar conceitos básicos da linguagem **Java**, como:

* Estruturas de repetição (`while`)
* Estruturas condicionais (`if / else`)
* Entrada de dados com `Scanner`
* Geração de números aleatórios com `Random`

# 🎯 Jogo de Adivinhação em Java

Projeto simples desenvolvido em **Java** como atividade do curso de **Desenvolvimento Back-End com Java** da Alura.

## 📌 Descrição

O programa simula um **jogo de adivinhação**, onde o sistema gera um número aleatório entre **0 e 50**.  
O usuário tem **5 tentativas** para acertar o número. A cada tentativa, o programa informa se o número digitado é **maior ou menor** que o número secreto.

## 🛠 Tecnologias utilizadas

- Java
- IntelliJ IDEA / VS Code
- Biblioteca `Random`
- Biblioteca `Scanner`
